  *(А. Горохов)*  
  
       ***C        F+7      G         C***  
  Она вышла из дома, не выключив свет,        ***F+7***  
       ***C         F+7     G     C          [+-+-+-+-+-+]***  
  Потеряв свой ключ и забыв о том.        [| | | | 0 |]  
     ***A            Em      A Em            [+-+-+-+-+-+]***  
  Купила цветы сама себе...               [| | | 0 | |]  
         ***F+7            C                 [+-+-+-+-+-+]***  
  Чтобы выбросить их потом.               [| | 0 | | |]  
                                          [+-+-+-+-+-+]  
  Ей нравилось быть такой,  
  Нравились фильмы, где герой погибает -  
  Умирает красивый герой.

  Дома долго звонил телефон,  
  ***C           F+7       G       C***  
  В дверь стучали, и ушли ни с чем.  
    ***Dm   C          G      C***  
  Птицы сели на пустой балкон,  
      ***Dm       G    C***  
  И, кажется, насовсем.
